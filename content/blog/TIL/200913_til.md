## JavaScript - Undefined..?! (1)  

오늘 [작심삼일](https://github.com/FacerAin/Jaksimsamil) 개발을 진행하는데 아래 코드가 이상하게 동작해서 애를 먹었다.  

```
export const getChallengeByUsername = (username) => {
  client.get(`api/challenge/getchallenge?username=${username}`); //Client 객체는 Axios.client 객체이다.
};
```  

분명 getChallengeByUsername 함수는 axios 객체를 반환해야하는데 자꾸 undefined만 반환하길래 한참 고민했다.

그러다 우연히 발견한 문제점을 발견하여 수정한 코드는 아래와 같다.  

```
export const getChallengeByUsername = (username) =>
  client.get(`api/challenge/getchallenge?username=${username}`); //Client 객체는 Axios.client 객체이다.
```
